# Changelog

All notable changes to Aurora Preview are documented here.

## [0.1.1] - 2026-07-06

### Changed

- New full-bleed aurora-gradient icon (bolder mark, no transparent corners).

## [0.1.0] - 2026-07-06

Initial release.

### Added

- Live Markdown preview to the side with bidirectional scroll-sync.
- **Native theming** that inherits the VS Code theme, plus `terminal-glass` and `sepia` presets.
- Customization settings: accent, reading font, code font, font size, line height, content width.
- Theme-aware syntax highlighting (highlight.js).
- KaTeX math (inline and block).
- Zero-setup Mermaid diagrams, themed from the accent, with click-to-zoom full-screen viewing.
- GitHub-style alert callouts (Note / Tip / Important / Warning / Caution).
- Task lists, tables, footnotes, frontmatter cards, code copy buttons, and an image lightbox.
- Export to a self-contained HTML file (CSS and fonts inlined) and to PDF via the print dialog.
